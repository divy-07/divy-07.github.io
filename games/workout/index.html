Installation Guide

Download

Click download above.
Click show in folder. (or open downloads folder)

Extract

Right click on workout.zip
Select extract all.



Choose a folder where you would like to install the program. (you can keep the default, which is downloads folder)
Then click Extract.

Open

Navigate to the folder you chose (if you kept default, it should be in downloads folder)
Open the workout folder and click exercise.exe.



If the program runs you are done. If you see a virus protection message, you'll need to turn off your antivirus to run this program.

I have outlined steps below if you get a message from Microsoft Defender.

Windows Defender




Click on more info and then select run anyway.



This should run the program, if not you'll need ot turn off any additional antivirus softwares you have.
